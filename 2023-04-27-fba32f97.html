搭建个人博客（通过hexo和gitee）

安装git

  1. 打开官方地址下载安装包
  2. 一路下一步
  3. 安装完成

配置git

将git与你的gitee账号绑定

  1. 鼠标在任何位置右键打开Git Blash

  1. 依次输入下列代码

git config --global user.name "你的Gitee用户名"

回车，绑定你的用户名

git config --global user.email "你的Gitee注册邮箱”

回车，绑定你的邮箱

生成SSH密钥

  1. 在Git Blash中输入以下代码以生成SSH密钥

ssh-keygen -t rsa

  1. 打开SSH密钥所在目录

C:\Users\你当前的用户名\.ssh

  1. 打开文件：id_rsa.pub，复制

配置SSH密钥

  1. 打开gitee主页，点击右上角头像，点击设置或者账号设置
  1. 在安全设置中选择SSH公钥

  1. 粘贴公钥到相应位置，确定

安装node.js

  1. 打开官方地址下载安装包

  2. 一路下一步

  3. 输入以下代码并返回版本号即为正确安装

    node -v v18.16.0

部署hexo环境

  1. 创建一个名为blog的文件，在文件里打开Git Bash
  2. 初始化hexo

hexo init

  1. 部署本地hexo网页

hexo s

  1. 按住control的同时点击网址即可打开，或者复制到浏览器打开

ps:如果打开网页错误，localhost拒绝访问，

你可以试试直接在浏览器输入127.0.0.1

到此你应该能看到nexo的默认网页了

将blog上传到gitee

在gitee上新建一个仓库

  1. 点击右上角加号-新建仓库，随便取个名称，选择开源（不然无法用Gitee Pages 发布网页），点击创建
  2. 进入创建好的仓库，点击服务-Gitee Pages，然后发布网页

  1. 你会在当前页面得到一个网址，它将是你以后访问博客的网址，复制这个网址

配置本地文件

  1. 打开blog文件夹下的_config.yml，将复制的网址填入url: 后（”:”后面有一个空格），并添加root信息

`url: 你复制的网址

root: /你创建的仓库名`

  1. 返回Gitee，打开你的仓库，点开橙色按钮：克隆/下载，复制https地址

  1. 继续编辑_config.yml文件，滑到最后找到deploy,修改为下列代码

deploy:
type: git
repository: 你复制的https地址
branch: main

  1. 安装hexo-deployer-git 自动部署发布工具，在gitee blash中执行

npm install hexo-deployer-git --save

  1. 生成页面

hexo g

  1. 上传文件

hexo d

此时放回你的Gitee仓库，你应该会看到内容更新

在浏览器中输入之前复制的网址，就可以访问你的网页了
